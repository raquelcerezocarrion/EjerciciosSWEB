Elementos básicos de HTML

  • <h1> a <h6>: Encabezados de diferentes niveles.
  • <p>: Define un párrafo de texto.
  • <a>: Crea un hipervínculo.
  • <img>: Inserta una imagen.
  • <ul> / <ol> + <li>: Listas.
  • <div> / <span>: Contenedores.
  • <form>, <input>, <button>: Formularios.
  • <table>, <tr>, <td>, <th>: Tablas.
  • <meta>, <link>, <script>, <style>: Cabecera/recursos.

Buenas prácticas en HTML

Usa etiquetas semánticas, indentación, alt en imágenes, evita exceso de div/span, separa CSS/JS en archivos, valida con W3C y prueba en navegadores/dispositivos.

Estructura básica

<!DOCTYPE html>
<html lang="es">
<head>...</head>
<body>...</body>
</html>

Ejemplo de estructura semántica

<header>...</header>
<main>...</main>
<footer>...</footer>

Ejemplo de uso de comentarios

<!-- Este es un comentario -->

Ejemplo de atributos globales

<div id="miDiv" class="miClase" style="color: red;" title="Div">Contenido</div>

Etiquetas de texto

<strong>Negrita</strong> <em>Cursiva</em> <mark>Resaltado</mark> ...
