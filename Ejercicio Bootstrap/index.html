Consejos básicos

Rutina rápida (5 minutos)

  1. Hidratante con SPF.
  2. Base donde lo necesites.
  3. Colorete en crema y cejas.
  4. Máscara de pestañas.
  5. Bálsamo o labial hidratante.

Trucos

  • Mezcla la base con hidratante para un acabado natural.
  • Iluminador en pómulo alto y lagrimal.
  • La barra de labios puede servir como rubor de emergencia.
